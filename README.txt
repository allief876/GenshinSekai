# GenshinSekai
Tugas Besar IF2121 Logika Komputasional 2020-01

## Cara Pakai
Dilakukan di Windows 10.
1. Pasang GNU Prolog di Komputer Anda. Installer dapat di-download disini: http://www.gprolog.org/#download
2. Download repository ini, di bagian Code (Pojok kanan atas) ada bagian untuk mengunduhnya.
3. Buka GNU Prolog, lalu lakukan change directory ke directory yang mengandung semua file program.
4. Jalankan "consult('main.pl')." tanpa tanda kutip untuk melakukan kompilasi.
5. Untuk mengetahui command-command yang bisa dipakai, jalankan "help." tanpa tanda kutip.
6. Jalankan "start." untuk memulai permainan.
7. Selamat bermain! Semoga menang.
